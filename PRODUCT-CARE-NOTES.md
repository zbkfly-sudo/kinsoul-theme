## Writing Principles

1. **Positive framing** — every tip tells the customer what TO DO, not what to avoid
2. **Material-specific** — each note mentions the actual stones in that bracelet by name
3. **Brand voice** — warm, personal, like advice from a knowledgeable friend
4. **"Last on, first off"** — this phrase appears in every note as a memorable takeaway
5. **Replacement cord** — mentioned in every note to reassure
6. **No technical jargon** — no Mohs hardness, no chemical terms, no scary warnings

## How to Apply in Shopify Admin

1. Go to **Products** → select the product
2. Scroll to **Metafields** section
3. Find **Care Note** (`custom.care_note`)
4. Paste the corresponding text (without the backtick fences)
5. Save
