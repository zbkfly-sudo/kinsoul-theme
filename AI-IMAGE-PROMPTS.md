## 使用说明

每张图提供了两个版本的 prompt：
- **`--v 7`** 版本：使用 V7 模型，真实感最强，推荐优先用这个
- **`--raw`** 版本：使用 V8 模型的 raw 模式，去除默认美化，输出更稳定直接

### 参数说明
| 参数 | 作用 |
|------|------|
| `--s 1000` | 最高风格化程度，让 Midjourney 发挥更多创意 |
| `--p` | 启用个性化，结合你的评分偏好出图 |
| `--ar 16:9` | 宽屏比例，适合 Hero 和背景图 |
| `--ar 4:5` | 竖版比例，适合产品图和模特图 |
| `--ar 1:1` | 正方形，适合包装图 |
| `--v 7` | 指定使用 V7 模型 |
| `--raw` | V8 raw 模式，去除默认风格化 |

### 出图流程
1. 先用普通模式快速生成预览（不加 `--hd`）
2. 看到满意的图后，点 Upscale 或加 `--hd` 重新生成 2K 大图
3. 下载后上传到 Shopify 后台：**内容** → **文件**
4. 然后在主题编辑器里选择对应图片
