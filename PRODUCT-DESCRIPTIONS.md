## Step-by-step: how to update each product

For each of the 6 products:

1. Shopify Admin → **Products**
2. Click the product
3. Find the **"Description / 描述"** field (usually right under the title)
4. **Select all** existing text and delete it
5. **Copy** the new description from the corresponding section above (just the text inside the ``` block)
6. **Paste** into the description field
